# FINPROPPT

TIKI Denpasar delivery prototype with:

- Laravel web app for customer and admin flows
- Go backend API for tracking, rates, and manifest simulation
- Expo mobile app prototype
- Vercel-compatible static frontend and Go serverless API

## Local URLs

- Laravel web: `http://127.0.0.1:8001/login`
- Go backend: `http://127.0.0.1:5000/health`
- Expo dev server: `http://localhost:8081`
